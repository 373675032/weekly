<https://github.com/sml2h3/ddddocr>
中国人不骗中国人，API的确很易用。
```python
import ddddocr

ocr = ddddocr.DdddOcr()

with open("test.jpg", 'rb') as f:
    image = f.read()

res = ocr.classification(image)
print(res)
```
